#  حالت‌های اصلی باز کردن فایل در پایتون

<table>
  <thead>
    <tr>
      <th>Mode</th>
      <th>توضیح</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>"r"</td>
      <td>خواندن (Read) — اگر فایل وجود نداشته باشد خطا می‌دهد</td>
    </tr>
    <tr>
      <td>"w"</td>
      <td>نوشتن (Write) — فایل را می‌سازد و اگر وجود داشته باشد پاک و بازنویسی می‌کند</td>
    </tr>
    <tr>
      <td>"a"</td>
      <td>افزودن (Append) — فایل را باز می‌کند و داده‌ها را به انتهای آن اضافه می‌کند</td>
    </tr>
    <tr>
      <td>"x"</td>
      <td>ایجاد (Create) — فقط اگر فایل وجود نداشته باشد ساخته می‌شود، وگرنه خطا می‌دهد</td>
    </tr>
  </tbody>
</table>

# حالت‌های ترکیبی با خواندن/نوشتن

<table>
  <thead>
    <tr>
      <th>Mode</th>
      <th>توضیح</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>"r+"</td>
      <td>خواندن + نوشتن (بدون حذف محتوا) — فایل باید وجود داشته باشد</td>
    </tr>
    <tr>
      <td>"w+"</td>
      <td>نوشتن + خواندن — فایل را پاک می‌کند یا می‌سازد</td>
    </tr>
    <tr>
      <td>"a+"</td>
      <td>افزودن + خواندن — هنگام خواندن، داده‌ها از اول فایل خوانده می‌شوند</td>
    </tr>
    <tr>
      <td>"x+"</td>
      <td>ایجاد + خواندن — اگر فایل وجود داشته باشد خطا می‌دهد</td>
    </tr>
  </tbody>
</table>

# حالت‌های باینری (Binary Mode)

<table>
  <thead>
    <tr>
      <th>Mode</th>
      <th>توضیح</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>"rb"</td>
      <td>خواندن باینری</td>
    </tr>
    <tr>
      <td>"wb"</td>
      <td>نوشتن باینری</td>
    </tr>
    <tr>
      <td>"ab"</td>
      <td>افزودن باینری</td>
    </tr>
    <tr>
      <td>"xb"</td>
      <td>ایجاد باینری</td>
    </tr>
    <tr>
      <td>"r+b" یا "rb+"</td>
      <td>خواندن + نوشتن باینری</td>
    </tr>
    <tr>
      <td>"w+b" یا "wb+"</td>
      <td>نوشتن + خواندن باینری</td>
    </tr>
    <tr>
      <td>"a+b" یا "ab+"</td>
      <td>افزودن + خواندن باینری</td>
    </tr>
    <tr>
      <td>"x+b" یا "xb+"</td>
      <td>ایجاد + خواندن باینری</td>
    </tr>
  </tbody>
</table>
